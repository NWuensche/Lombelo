## Tools
* Spring Boot as the MVC tool
* Bootstrap + Thymeleaf to create Views

## Build Lombelo from Source
* If you don't haven maven, you have to install it.
* In the main folder, execute maven:

``bash
$ mvn spring-boot:run
``
